# Chinese Color Cursors

Pequeña página en HTML para practicar vocabulario de colores en chino. Cada palabra muestra un cursor personalizado con un cuadrito y emojis de números para gamificar la práctica.

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Pasa el cursor por encima de cada palabra de color para ver el cuadrito con el número-emoji correspondiente.

## GitHub Pages
Si tienes GitHub Pages activado en el repositorio, la página quedará disponible en:

```
https://<tu-usuario>.github.io/ChineseColorCursors/
```

Sustituye `<tu-usuario>` por tu nombre de usuario de GitHub. Si tu configuración de Pages usa otra rama o carpeta (por ejemplo, `/docs`), ajusta la ruta según corresponda.

## Personalización rápida
- Cambia los textos o emojis en `index.html` para adaptarlos a tu estilo de enseñanza.
- Si añades más colores, duplica un bloque `<p>` y actualiza la clase del `span` junto con la descripción.
